# Task 1: Network Scanning using Nmap

## Objective
To scan the local network and identify active hosts and open ports using Nmap.

## Tool Used
- Nmap (TCP SYN Scan)

## Network Range
192.168.29.0/24

## Scan Command
nmap -sS 192.168.29.0/24

## Outcome
- Discovered multiple active hosts
- Identified open ports and running services
- Analyzed potential security risks

## Note
Scan was performed only on my personal local network for educational purposes.
